# 🎯 QR Generator Pro

Professional QR Code Generator - Run instantly with NPX!

## 🚀 Usage

```bash
npx qr-generator-pro
```

## ✨ Features

- 🎯 Generate QR codes for any content
- 🎨 Custom colors and sizes
- 📱 Real-time preview
- 💾 Download as PNG
- 🌐 Works offline
- ⚡ Instant setup with NPX

## 📱 Supported Content Types

- Website URLs
- Plain text
- Email addresses
- Phone numbers
- WiFi credentials
- And more!
